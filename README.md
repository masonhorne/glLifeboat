# glLifeboat
[![NPM version](https://badge.fury.io/js/gllifeboat.svg)](http://badge.fury.io/js/gllifeboat)
[![npm](https://img.shields.io/npm/dm/gllifeboat.svg?maxAge=2592000)](https://npmcharts.com/compare/gllifeboat?minimal=true)

A javascript library for rendering to the WebGL Canvas.

Install the latest version of glLifeboat from [npm](https://www.npmjs.com/package/gllifeboat).

```bash
npm install gllifeboat
```
